# Conditions générales d'utilisation

Ces conditions sont de la société Chronos-Travel GmbH.

Elles concernent la location d'appareils de voyage dans le temps.

---

## Partie 1 : L'objet du contrat et les risques

La société Chronos-Travel GmbH loue un appareil.

Nous appelons cette société : le **loueur**.

Cela signifie :

La personne qui donne l'appareil.

Vous louez l'appareil.

Nous vous appelons : le **locataire**.

Cela signifie :

La personne qui utilise l'appareil.

L'appareil s'appelle : Flux-Master 3000.

Ou un appareil similaire.

Cet appareil permet de voyager dans le temps.

---

### Les risques

Cet appareil est dangereux.

Il peut vous blesser.

Il peut aussi endommager l'espace et le temps.

On appelle cela : le **continuum espace-temps**.

Cela signifie :

L'ensemble de l'espace et du temps.

---

### Ce qui est interdit

Vous ne pouvez pas changer le passé.

Par exemple : empêcher la naissance de vos grands-parents.

On appelle cela : le **paradoxe du grand-père**.

Cela signifie :

Si vous empêchez la naissance de votre grand-père,
vous ne pouvez plus exister.
Mais alors, qui a empêché sa naissance ?

C'est un problème logique.

C'est **strictement interdit**.

---

## Partie 2 : Vos obligations

### Comment utiliser l'appareil

Vous devez suivre le mode d'emploi.

Lisez bien les instructions.

### Ce qui est interdit

Vous ne pouvez pas utiliser l'appareil :
- Si vous avez bu de l'alcool
- Si vous avez pris des drogues
- Si vous êtes très émotif

Par exemple :
- Si vous avez le cœur brisé
- Si vous voulez vous venger d'une personne historique

---

### Interdiction de s'enrichir

Vous ne pouvez pas apporter dans le passé :
- Des résultats sportifs du futur
- Des informations sur la bourse
- Des numéros de loterie gagnants

Vous ne pouvez pas utiliser ces informations pour gagner de l'argent.

On appelle cette règle : la **Directive éthique du voyage dans le temps**.

En allemand : **Zeitreise-Ethik-Richtlinie (ZER)**.

Cela signifie :

Les règles pour voyager dans le temps de manière éthique.

Si vous ne respectez pas cette règle :
- Le contrat est annulé immédiatement
- Vous payez une amende de 50 000 crédits

---

## Partie 3 : Responsabilité en cas de problèmes avec le temps

### Si vous disparaissez

Vous pouvez faire des erreurs dans le passé.

Ces erreurs peuvent empêcher votre propre naissance.

Vous disparaissez alors.

Le loueur n'est pas responsable de cela.

---

### Si vous restez coincé dans une boucle

Vous pouvez rencontrer votre ancien vous-même.

Cela peut créer une **boucle causale**.

Cela signifie :

Une boucle où le même jour se répète sans fin.

C'est comme dans le film : Un jour sans fin.

On appelle cela aussi : le **scénario de la marmotte**.

Si cela vous arrive :
- Vous aurez peut-être besoin d'un psychologue après
- Le loueur ne paie pas ce psychologue

---

### L'effet papillon

Vous pouvez faire de petites actions dans le passé.

Ces petites actions peuvent avoir de grandes conséquences.

On appelle cela : l'**effet papillon**.

Cela signifie :

Un petit changement peut causer un grand changement.

Par exemple :
Vous écrasez un insecte il y a des millions d'années.
Cela peut empêcher l'existence de l'humanité.

Si cela arrive :
- Le loueur rembourse au maximum votre caution
- Pas plus
- Seulement si la loi le permet

---

## Partie 4 : Le retour de l'appareil

Vous devez rendre l'appareil.

Vous devez le rendre dans la même **ligne temporelle**.

Cela signifie :

Le même univers d'où vous l'avez pris.

### Ce qui n'est pas accepté

Vous ne pouvez pas rendre l'appareil dans un autre univers.

Par exemple : un univers où des hommes-lézards intelligents gouvernent le monde.

Cela ne compte pas comme un retour valide.
