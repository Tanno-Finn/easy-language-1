# Directive: Проста Мова (Ukrainian Plain Language) [M]

## Basis [M]
**Type:** Adaptation
**Framework:** Based on Plain Language principles with Ukrainian grammar adaptations
**Origin:** Adapted for Ukrainian; no established Ukrainian Plain Language standard exists

---

## Purpose [M]
Translate text into **Проста Мова** (Plain Ukrainian).

## Language Settings [M]
- **Output Language:** Ukrainian
- **Anchor Phrase:** "**Іншими словами:**"
- **Level:** B1 (CEFR)

---

## Core Rules [M]

### 1. Sentence Structure [M]
- Maximum 15-20 words per sentence.
- Prefer Subject-Verb-Object order.
- Limit subordinate clauses to one per sentence.
- One main idea per sentence.

### 2. Technical Terms [M]
When you encounter a technical term:

1. Keep the term.
2. Add brief explanation in parentheses OR
3. Add "**Іншими словами:**" followed by explanation.

**Example:**
```
Ми обробляємо ваші персональні дані (інформацію про вас, наприклад ім'я та адресу).
```

OR:

```
Ми обробляємо ваші персональні дані.

Іншими словами:

Це інформація про вас, наприклад ім'я та адреса.
```

### 3. Grammar Guidelines [M]
**Prefer active voice:**
- **Avoid:** "Дані обробляються нами"
- **Prefer:** "Ми обробляємо дані"

**Limit participle constructions:**
- Use sparingly, only when clear.
- Break complex constructions into separate sentences if needed.

**Keep genitive chains short:**
- **Avoid:** "Результати перевірки документів заявника"
- **Prefer:** "Результати перевірки. Ми перевірили документи заявника."

### 4. Vocabulary [M]
- Use clear, everyday Ukrainian words.
- Avoid jargon where possible.
- Expand abbreviations on first use.
- Use "ви" (formal you) for addressing the reader.

### 5. Formatting [M]
- Use bullet points for lists of 3+ items.
- Use headings to structure longer texts.
- Numbers as digits for amounts, dates.
- Bold key terms when helpful.

## QA-Log Section [M]

At the end of every translation, append:

```
---
### QA-лог (Згенеровано автоматично)

**Пояснені терміни:**
- [list terms with explanations added]

**Розшифровані скорочення:**
- [list abbreviations expanded]

**Змінені конструкції:**
- [list constructions simplified]
```

## Example Output [M]

**Input:**
> Персональні дані обробляються відповідно до чинного законодавства про захист персональних даних з урахуванням вимог щодо конфіденційності.

**Output:**
```
Ми обробляємо ваші персональні дані (інформацію про вас).
Ми дотримуємося закону про захист даних.
Ваша інформація залишається конфіденційною.
```
