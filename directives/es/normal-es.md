# Directiva: Traducción Profesional al Español [M]

## Meta [M]
| Campo | Valor |
|-------|-------|
| **ID** | `normal-es` |
| **Tipo** | Traducción Profesional Estándar |
| **Versión** | 2.0 |
| **Dirección del Texto** | LTR (Izquierda a Derecha) |
| **Nivel Objetivo** | Nativo / C1-C2 |

---

## Propósito [M]

Traducir textos fuente a un **español natural y profesional**, manteniendo precisión, registro y adecuación cultural.

---

## Cuándo Usar [M]

- Documentos profesionales que requieren traducción precisa
- Textos comerciales, legales o técnicos
- Contenido donde la fluidez natural es primordial
- Cuando NO se requiere Lectura Fácil (A1/A2) ni Lenguaje Claro (B1)

---

## Configuración del Idioma [M]

### Idioma de Salida
- **Principal:** Español de España (por defecto) o Español de Latinoamérica (especificar en la solicitud)
- **Escritura:** Latina
- **Dirección del Texto:** LTR

### Idiomas Fuente Aceptados [M]
- Alemán (principal)
- Cualquier idioma (con solicitud explícita del usuario)

### Manejo de Escrituras [M]
| Escritura Fuente | Manejo |
|-----------------|--------|
| Latina (DE, FR, EN...) | Traducción directa |
| Cirílica (RU) | Transliterar nombres si es necesario |
| CJK (ZH, JA, KO) | Romanizar nombres propios; mantener marcas |
| Árabe/Hebreo (RTL) | Asegurar incrustación correcta en salida LTR |

---

## Principios Fundamentales [M]

### 1. Precisión Primero [M]
- Preservar TODA la información del texto fuente
- NO añadir, omitir ni inventar contenido
- Mantener validez legal/técnica de las declaraciones
- **NUNCA inventar datos** [M]

### 2. Términos Técnicos [M]
- Usar equivalentes españoles establecidos para términos técnicos
- Si NO existe equivalente estándar: mantener término original + explicación en primer uso
- **Regla de Consistencia:** Mismo término = misma traducción en todo el documento [M]

**Formato para términos desconocidos:**
```
[Término Original] (explicación en español)
```

### 3. Español Natural [M]
- Usar español idiomático, NO traducción literal
- Adaptar estructura oracional a convenciones españolas
- Evitar patrones sintácticos del idioma fuente

### 4. Correspondencia de Registro [M]
| Registro Fuente | Registro Objetivo |
|----------------|-------------------|
| Formal/Legal | Español formal |
| Comercial | Español profesional |
| Técnico | Español técnico |
| Informal | Español coloquial |

**Manejo de Pronombres:**
- Alemán "Sie" (formal) → "usted/ustedes"
- Alemán "du" (informal) → "tú/vosotros" (España) o "vos" (Latinoamérica, si aplica)

---

## Manejo de Elementos Específicos [M]

### A. Palabras Compuestas [M]
Transformar compuestos del idioma fuente en español natural:

| Alemán | Español |
|--------|---------|
| Datenschutzerklärung | política de privacidad |
| Nutzungsbedingungen | condiciones de uso |
| Haftungsausschluss | exención de responsabilidad |
| Verantwortungsbereich | área de responsabilidad |

**Regla:** Si el compuesto es específico del dominio, verificar que existe equivalente español estándar.

### B. Referencias Legales [M]
| Fuente | Objetivo |
|--------|----------|
| § | Artículo / Sección |
| Abs. | Apartado / Párrafo |
| Nr. | Número / Punto |

- Mantener números de párrafo/artículo intactos
- Indicar diferencias jurisdiccionales si es legalmente relevante

### C. Números y Formatos [M]

#### Convenciones Numéricas
| Elemento | Formato Español |
|----------|----------------|
| Miles | 1.000 (punto) |
| Decimales | 0,50 (coma) |
| Porcentajes | 50 % o 50 por ciento |
| Teléfono | +34 91 123 45 67 |

#### Formatos de Fecha
| Variante | Formato | Ejemplo |
|----------|---------|---------|
| España | DD de mes de AAAA | 15 de enero de 2024 |
| Latinoamérica | DD de mes de AAAA | 15 de enero de 2024 |
| ISO (neutral) | AAAA-MM-DD | 2024-01-15 |

**Regla:** Usar formato de la audiencia objetivo o ISO para textos internacionales.

#### Moneda
- Mantener moneda original con símbolo: 50 €, 30 £, 100 $
- Añadir equivalente entre paréntesis si es relevante: 50 € (aprox. 55 USD)

### D. Abreviaturas [M]
Expandir abreviaturas del idioma fuente y usar equivalentes españoles:

| Fuente | Español |
|--------|---------|
| z.B. | p. ej. / por ejemplo |
| d.h. | es decir |
| u.a. | entre otros |
| etc. | etc. (mantener) |
| usw. | etc. |
| ggf. | en su caso / si procede |
| bzgl. | respecto a / en relación con |
| inkl. | incluido/a |
| exkl. | excluido/a |
| e.g. | p. ej. |
| i.e. | es decir |

### E. Puntuación [M]
| Elemento | Regla Española |
|----------|---------------|
| Comillas | «comillas angulares» (preferido) o "comillas inglesas" |
| Listas | Sin coma de Oxford en español |
| Dos puntos | Minúscula después de dos puntos (salvo inicio de cita) |
| Guiones | Guion largo (—) para incisos |
| Signos de interrogación/exclamación | ¿? y ¡! siempre |

## Género Gramatical [M]

### Reglas de Concordancia [M]
- Sustantivos: concordancia de género y número
- Adjetivos: concordar con el sustantivo en género y número
- Artículos: el/la/los/las según corresponda

### Lenguaje Inclusivo [C]
Si el cliente lo solicita, opciones disponibles:
- Desdoblamiento: "los usuarios y las usuarias"
- Sustantivos colectivos: "el personal" en lugar de "los empleados"
- Estructuras neutras: "quienes usen" en lugar de "los que usen"

**Nota:** Por defecto, usar masculino genérico según la RAE, salvo indicación contraria.

---

## Adaptación Cultural [M]

### Modismos y Metáforas [M]
- NO traducir literalmente
- Encontrar modismo español equivalente O reformular de manera neutra

| Modismo Alemán | NO | EN SU LUGAR |
|----------------|-----|-------------|
| Das ist nicht mein Bier | Eso no es mi cerveza | Eso no es asunto mío |
| Tomaten auf den Augen haben | Tener tomates en los ojos | No ver lo evidente |

### Referencias Culturales [C]
- Localizar si la audiencia objetivo no entenderá
- Mantener original + explicación para textos técnicos/legales

**Ejemplo:**
> "...nach dem BGB" → "...conforme al Código Civil alemán (BGB)"

---

## Integración de Glosarios [M]

### Orden de Prioridad [M]
1. **Glosario del cliente** (si se proporciona) — usar términos EXACTOS
2. **Glosario del dominio** (si existe en `glossaries/`)
3. **Traducción estándar** (criterio profesional)

### Regla de Prevalencia del Glosario [M]
Si el glosario contiene un término: **USARLO TAL CUAL**.
NO inventar alternativas cuando existe una definición.

---

## Manejo de Errores [M]

### Elementos Intraducibles [M]
| Situación | Acción |
|-----------|--------|
| Acrónimo sin contexto | Mantener original, añadir nota [?] |
| Referencia faltante | Marcar con [SE REQUIERE REFERENCIA] |
| Término ambiguo | Traducir significado más probable, anotar en QA-Log |
| Texto corrupto | Marcar como [ILEGIBLE] |

### Resolución de Conflictos [M]
- Precisión de la fuente > fluidez del objetivo
- En caso de duda: marcar y preguntar

---

## Especificaciones de Salida [M]

### Formato [M]
- Mantener estructura del documento fuente (encabezados, párrafos, listas)
- Preservar marcadores de formato si están presentes
- Salida de texto limpia (sin notas del traductor en línea)

### Nomenclatura de Archivos [M]
```
{nombre_original}_{lang}.txt
```
Ejemplo: `agb_zeitreise_es.txt`

## Diferencias España vs. Latinoamérica [C]

| Aspecto | España | Latinoamérica |
|---------|--------|---------------|
| Pronombre informal | vosotros | ustedes |
| Pasado reciente | Pretérito perfecto | Pretérito indefinido |
| Vocabulario | ordenador, móvil | computadora, celular |

**Regla:** Por defecto usar español de España. Adaptar a Latinoamérica solo si se especifica.
