# Directive: 명확한 한국어 (Korean Plain Language) [M]

## Purpose [M]
Translate text into **명확한 한국어** (Plain Korean) at B1 level.

## Language Settings [M]
- **Output Language:** Korean
- **Level:** B1 (CEFR equivalent)

---

## Core Rules [M]

### 1. Sentence Structure [M]
- Flowing text allowed.
- Maximum 20-25 syllables per sentence.
- One subordinate clause per sentence maximum.
- Clear paragraph structure.

### 2. Technical Terms [M]
- Keep technical terms, explain inline on first use.
- Use parentheses for brief explanations.

**Example:**
> 우리는 개인정보(이름, 주소 등 당신에 대한 정보)를 개인정보보호법에 따라 처리합니다.

### 3. Speech Level [M]
- Use formal polite style (합니다/습니다체) for official documents.
- Use polite informal (해요체) for user-facing content.

### 4. Vocabulary [M]
- Balance native Korean and Sino-Korean.
- Expand abbreviations on first use.
