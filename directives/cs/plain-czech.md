# Directive: Srozumitelná Čeština (Czech Plain Language) [M]

## Purpose [M]
Translate text into **Srozumitelná Čeština** (Plain Czech) at B1 level.

## Language Settings [M]
- **Output Language:** Czech
- **Level:** B1 (CEFR equivalent)

---

## Core Rules [M]

### 1. Sentence Structure [M]
- Flowing text allowed.
- Maximum 15-20 words per sentence.
- One subordinate clause per sentence maximum.
- Clear paragraph structure.

### 2. Technical Terms [M]
- Keep technical terms, explain inline on first use.
- Use parentheses for brief explanations.

**Example:**
> Zpracováváme vaše osobní údaje (informace o vás, jako je jméno a adresa) v souladu se zákonem o ochraně osobních údajů (GDPR).

### 3. Grammar [M]
- Simplify case usage where natural.
- Prefer active voice.
- Avoid unnecessary complexity.

### 4. Vocabulary [M]
- Common Czech vocabulary.
- Expand abbreviations on first use.
