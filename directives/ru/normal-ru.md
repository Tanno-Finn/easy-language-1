## Purpose [M]

Translate source texts into **natural, professional Russian** while maintaining accuracy, register, and cultural appropriateness.

---

## When to Use [M]

- Professional documents requiring accurate translation
- Business, legal, or technical texts
- Content where natural fluency is paramount
- When Easy Read (A1/A2) or Plain Language (B1) is NOT required

---

## Language Settings [M]

### Output Language
- **Primary:** Standard Russian (Современный русский литературный язык)
- **Script:** Cyrillic
- **Text Direction:** LTR

### Accepted Source Languages [M]
- German (primary)
- English
- Any language (with explicit user request)

### Script Handling [M]
| Source Script | Handling |
|--------------|----------|
| Latin (DE, EN, FR...) | Transliterate proper nouns per GOST 7.79-2000 or keep original if widely known |
| Cyrillic (UK, BG, SR) | Direct translation |
| CJK (ZH, JA, KO) | Transliterate per established conventions; keep brand names |
| Arabic/Hebrew (RTL) | Transliterate proper nouns |

## Core Principles [M]

### 1. Accuracy First [M]
- Preserve ALL information from the source text
- Do NOT add, omit, or invent content
- Maintain legal/technical validity of statements
- **NEVER hallucinate facts** [M]

### 2. Technical Terms [M]
- Use established Russian equivalents for technical terms
- If NO standard equivalent exists: keep original term + explanation on first use
- **Consistency Rule:** Same term = same translation throughout [M]

**Format for unknown terms:**
```
[Original Term] (русское объяснение)
```

### 3. Natural Russian [M]
- Use idiomatic Russian, NOT word-for-word translation
- Adapt sentence structure to Russian conventions
- Avoid source-language syntax patterns
- Leverage Russian's flexible word order for emphasis and flow

### 4. Register Matching [M]
| Source Register | Target Register |
|----------------|-----------------|
| Formal/Legal | Официально-деловой стиль |
| Business | Деловой русский |
| Technical | Научно-технический стиль |
| Casual | Разговорный стиль |

**Pronoun Handling:**
- Source formal (German "Sie", English formal) → "Вы" (capitalized for respect)
- Source informal (German "du") → "ты"
- Generic/neutral → "вы" (lowercase, plural/generic)

---

## Grammatical Gender [M]

Russian has three genders: masculine, feminine, neuter.

### Agreement Rules [M]
- Adjectives MUST agree with nouns in gender, number, and case
- Past tense verbs agree with subject gender
- Participles and pronouns follow the same agreement

### Handling Gender-Unknown Referents [M]
| Situation | Approach |
|-----------|----------|
| Unknown person (singular) | Use masculine default OR restructure sentence |
| Mixed group | Use masculine plural (traditional) |
| Inclusive alternative | Use plural forms when possible |

## Handling Specific Elements [M]

### A. Compound Words [M]
German/English compounds typically become phrases or derived words in Russian:

| German | English | Russian |
|--------|---------|---------|
| Datenschutzerklärung | privacy policy | политика конфиденциальности |
| Nutzungsbedingungen | terms of use | условия использования |
| Haftungsausschluss | disclaimer | отказ от ответственности |
| Verantwortungsbereich | area of responsibility | сфера ответственности |

**Rule:** Verify established Russian terminology in legal/technical domains.

### B. Legal References [M]
| Source | Russian |
|--------|---------|
| § | статья / пункт |
| Section | раздел |
| Article | статья |
| Paragraph | пункт / абзац |
| Subsection | подпункт |

- Keep paragraph/article numbers intact
- Note jurisdiction differences: "согласно Гражданскому кодексу Германии (BGB)"

### C. Numbers and Formats [M]

#### Number Conventions
| Element | Russian Format |
|---------|---------------|
| Thousands | 1 000 (non-breaking space) |
| Decimals | 0,50 (comma) |
| Percentages | 50% or 50 процентов |
| Phone | +7 (495) 123-45-67 |

#### Date Formats
| Context | Format | Example |
|---------|--------|---------|
| Standard | DD.MM.YYYY | 15.01.2024 |
| Formal/narrative | DD месяц YYYY г. | 15 января 2024 г. |
| ISO (technical) | YYYY-MM-DD | 2024-01-15 |

**Month names:** января, февраля, марта, апреля, мая, июня, июля, августа, сентября, октября, ноября, декабря (genitive case for dates)

#### Currency
- Keep original currency with symbol: 50 €, 30 £, 100 $
- Russian rubles: 1 000 ₽ or 1 000 руб.
- Add equivalent if relevant: 50 € (примерно 5 000 ₽)

### D. Abbreviations [M]
Expand source abbreviations and use Russian equivalents:

| Source | Russian |
|--------|---------|
| e.g. / z.B. | например / напр. |
| i.e. / d.h. | то есть / т. е. |
| etc. / usw. | и так далее / и т. д. |
| among others / u.a. | в том числе / в т. ч. |
| if applicable / ggf. | при необходимости |
| regarding / bzgl. | относительно / касательно |
| including / inkl. | включая / вкл. |
| excluding / exkl. | исключая / за исключением |

### E. Punctuation [M]
| Element | Russian Rule |
|---------|-------------|
| Quotation marks | «ёлочки» (primary), „лапки" (nested) |
| Dash | — (em-dash with spaces) for clauses |
| Lists | No Oxford comma (not used in Russian) |
| Decimals | Comma (0,5) not period |
| Thousands | Non-breaking space (1 000) |

**Quotation example:**
```
Он сказал: «Прочитайте раздел „Условия использования"».
```

## Case System [M]

Russian has six grammatical cases. Ensure correct case usage:

| Case | Function | Preposition Examples |
|------|----------|---------------------|
| Именительный | Subject | — |
| Родительный | Possession, "of" | без, для, до, из, от, у |
| Дательный | Indirect object, "to" | к, по |
| Винительный | Direct object | в, на, за, через |
| Творительный | Instrument, "with/by" | за, между, над, под, с |
| Предложный | Location, "about" | в, на, о, при |

**Critical:** Verify case endings for proper nouns and technical terms.

## Cultural Adaptation [M]

### Idioms and Metaphors [M]
- Do NOT translate literally
- Find equivalent Russian idiom OR rephrase neutrally

| German Idiom | NOT | INSTEAD |
|--------------|-----|---------|
| Das ist nicht mein Bier | Это не моё пиво | Это не моя забота |
| Tomaten auf den Augen haben | Иметь помидоры на глазах | Не замечать очевидного |

### Cultural References [C]
- Localize if target audience won't understand
- Keep original + explanation for technical/legal texts

**Example:**
> "...nach dem BGB" → "...в соответствии с Гражданским кодексом Германии (BGB)"

### Name Transliteration [M]
| Type | Approach |
|------|----------|
| Personal names | Transliterate (Müller → Мюллер) unless established form exists |
| Company names | Keep original Latin script OR use established Russian form |
| Place names | Use official Russian names (München → Мюнхен) |

## Glossary Integration [M]

### Priority Order [M]
1. **Client glossary** (if provided) — use EXACT terms
2. **Domain glossary** (if exists in `glossaries/`)
3. **Standard translation** (your professional judgment)

### Glossary Override Rule [M]
If glossary contains a term: **USE IT VERBATIM**.
Do NOT invent alternatives when a definition exists.

## Error Handling [M]

### Untranslatable Elements [M]
| Situation | Action |
|-----------|--------|
| Acronym without context | Keep original, add [?] note |
| Missing reference | Flag with [ТРЕБУЕТСЯ ССЫЛКА] |
| Ambiguous term | Translate most likely meaning, note in QA-Log |
| Corrupted text | Mark as [НЕЧИТАЕМО] |

### Conflict Resolution [M]
- Source accuracy > target fluency
- If unsure: flag and ask

### Format [M]

