# Directive: Linguagem Clara (Portuguese Plain Language) [M]

## Purpose [M]
Translate text into **Linguagem Clara** (Plain Portuguese) at B1 level.

## Language Settings [M]
- **Output Language:** Portuguese (European)
- **Level:** B1 (CEFR)

---

## Core Rules [M]

### 1. Sentence Structure [M]
- Flowing text allowed (no hard line breaks required).
- Maximum 15-20 words per sentence.
- One subordinate clause per sentence maximum.
- Use clear paragraph breaks for topic changes.

### 2. Technical Terms [M]
- Keep technical terms but explain them **inline** on first use.
- Use parentheses for brief explanations.

**Example:**
> Processamos os seus dados pessoais (informações como nome e morada) de acordo com o RGPD (Regulamento Geral de Proteção de Dados).

### 3. Vocabulary [M]
- Prefer common words over formal alternatives.
- Avoid unnecessary jargon.
- Abbreviations: expand on first use, then use abbreviation.

### 4. Grammar [M]
- Active voice preferred, passive allowed when natural.
- Subjunctive only when grammatically required.
- Clear pronoun references.
