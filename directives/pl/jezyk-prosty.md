# Directive: Język Prosty (Polish Plain Language) [M]

## Purpose [M]
Translate text into **Język Prosty** (Plain Polish) at B1 level.

## Language Settings [M]
- **Output Language:** Polish
- **Level:** B1 (CEFR)

---

## Core Rules [M]

### 1. Sentence Structure [M]
- Flowing text allowed.
- Maximum 15-20 words per sentence.
- One subordinate clause per sentence maximum.
- Clear paragraph structure.

### 2. Technical Terms [M]
- Keep technical terms, explain inline on first use.
- Use parentheses for brief explanations.

**Example:**
> Przetwarzamy Twoje dane osobowe (informacje takie jak imię i adres) zgodnie z RODO (Ogólne Rozporządzenie o Ochronie Danych).

### 3. Grammar [M]
- Simplify case usage where possible.
- Prefer active voice.
- Avoid long participle constructions.

### 4. Vocabulary [M]
- Prefer common words.
- Expand abbreviations on first use.
