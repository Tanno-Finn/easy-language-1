# Directive: Easy Indonesian (Bahasa Indonesia Mudah)

## Basis [M]
**Type:** Cognitive Accessibility Standard
**Target Level:** A2-B1 equivalent
**Script:** Latin
**Origin:** Indonesian adaptation of international Easy-to-Read principles

---

## Purpose [M]
Defines rules for translating texts into Easy Indonesian while maintaining accuracy (Hybrid Approach).

**Priority:** Accuracy over maximum simplification. [M]
**Approach:** Preserve technical terms, explain them clearly.

## When to Use [M]
- For texts targeting people with cognitive disabilities [M]
- For legally relevant documents (contracts, policies) [M]
- For new Indonesian language learners
- When maximum clarity is required

---

## Key Requirements [M]

### The Hybrid Rule (Supreme Directive) [M]
Technical terms are **preserved** and explained.

#### 1. Glossary Priority [M]
If the glossary contains a term with an explanation:
**You MUST use this explanation word-for-word.**
Do not invent new explanations when a definition exists.

#### 2. The Anchor Phrase [M]
Use **exclusively** the phrase "Artinya:" for explanations.

#### 3. Anti-Doubling (CRITICAL) [M]
If the original sentence contains words before the technical term like "artinya:", "berarti:", "adalah:", "yaitu:" -> **DELETE these words** before adding "Artinya:".

#### 4. Hard Line Break (Mandatory) [M]
Every explanation needs **three separate paragraphs**:
1. Technical term (**bold**) + full stop.
2. New paragraph -> "Artinya:" (alone).
3. New paragraph -> Explanation.

**Correct Format:**
> **Efisiensi**.
>
> Artinya:
>
> Berapa banyak sumber daya yang Anda butuhkan untuk mencapai tujuan.

**Incorrect Format:**
> Efisiensi artinya berapa banyak sumber daya yang dibutuhkan.

## Grammar & Sentence Structure [M]

### Word Order [M]
- Use **SVO order** (Subject - Verb - Object) [M]
- Indonesian naturally uses SVO, maintain this structure

**Example:**
> Good: Ibu membeli sayuran. (Mother buys vegetables.)
> Avoid: Sayuran dibeli oleh ibu. (Vegetables are bought by mother.)

### Active Voice [M]
- **Prefer active voice** (me- prefix) over passive (di- prefix) [M]
- Passive voice is common in Indonesian but harder to understand

**Example:**
> Good: Dokter memeriksa pasien. (The doctor examines the patient.)
> Avoid: Pasien diperiksa oleh dokter. (The patient is examined by the doctor.)

### Sentence Length [M]
- **One sentence = One idea** [M]
- Maximum 12-15 words per sentence
- Break complex sentences into multiple simple sentences

### Affixes [C]
- Indonesian uses many affixes (me-, ber-, di-, ke-an, -kan, -i)
- Prefer simpler base words when possible
- If affixed words are clearer, use them

**Example:**
> Simpler: Dia jalan ke toko. (He walks to the store.)
> Standard: Dia berjalan ke toko. (He walks to the store.)
> Use whichever is clearer in context.

---

## Formatting [M]

### Line Breaks [M]
- Every sentence starts on a **new line** [M]
- Use many paragraphs [M]
- One thought per paragraph

### Lists [C]
- Use bullet points for lists
- Maximum 5 items per list
- Each item should be short

---

## Edge Cases & Special Rules [M]

### A. Complex Words [M]
Replace formal words with everyday equivalents:
| Formal | Easy |
|--------|------|
| mengkonsumsi | makan, minum |
| mengakuisisi | membeli |
| mengimplementasikan | melakukan |
| komprehensif | lengkap |
| signifikan | penting |

### B. Abbreviations [M]
**Always write them out** on first use:
| Abbreviation | Full Form |
|--------------|-----------|
| dll. | dan lain-lain |
| dsb. | dan sebagainya |
| yg | yang |
| dg | dengan |
| utk | untuk |

### C. Loanwords (Foreign Words) [M]
Indonesian has many loanwords. Explain them if unclear:
> **Deadline**.
>
> Artinya:
>
> Batas waktu untuk menyelesaikan sesuatu.

### D. Numbers and Dates [M]
- Write numbers as **digits** (1, 2, 3, not satu, dua, tiga)
- Exception: "satu" and "dua" are acceptable for small counts
- Write dates clearly: 1 April 2024

### E. Reduplication [C]
Indonesian uses reduplication (anak-anak, buku-buku).
- Use reduplication normally, it's part of the language
- Explain if meaning might be unclear

---

## Pronouns [C]

Use clear, respectful pronouns:
| Context | Pronoun |
|---------|---------|
| Formal "you" | Anda |
| Reader addressing | Anda |
| "We" inclusive | kita |
| "We" exclusive | kami |

Avoid: lu, gue, lo (informal Jakarta slang)
