# Directive: 通俗中文 (Chinese Plain Language) [M]

## Purpose [M]
Translate text into **通俗中文** (Plain Chinese) at B1 level.

## Language Settings [M]
- **Output Language:** Simplified Chinese (简体中文)
- **Level:** B1 (CEFR equivalent)

---

## Core Rules [M]

### 1. Sentence Structure [M]
- Flowing text allowed.
- Maximum 25-30 characters per sentence.
- One subordinate clause per sentence maximum.
- Clear paragraph structure.

### 2. Technical Terms [M]
- Keep technical terms, explain inline on first use.
- Use parentheses for brief explanations.

**Example:**
> 我们处理您的个人数据（关于您的信息，如姓名和地址），遵守数据保护法（保护您信息的法律）。

### 3. Vocabulary [M]
- Prefer common modern Chinese.
- Chengyu allowed if commonly known.
- Expand abbreviations on first use.

### 4. Style [M]
- Balance between spoken and written style.
- Clear and direct expression.
